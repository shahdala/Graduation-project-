# Graduation-project-
Machine Learning and Deep Learning models for liver disease detection, fatty liver classification, and hepatocellular carcinoma (HCC) diagnosis using clinical data and ultrasound images.
## Models Included
- Liver Disease Detection (ML – clinical data)
- Fatty Liver Detection (ML – lab data)
- HCC Classification (CNN – ultrasound images)

These models were developed as part of a graduation project focused on AI-assisted liver disease diagnosis.
